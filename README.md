# event-management-dashboard
A dynamic event management dashboard that enables users to create, categorize, and manage events in real-time. Built using vanilla JavaScript with DOM manipulation and local data handling

Smart Event Dashboard is a front-end web application built using HTML, CSS, and JavaScript. 

Features:
- Add new events with title, date, category, and description
- Dynamic event listing using DOM manipulation
- Clear all events functionality
- Add sample events feature
- Interactive UI with responsive card layout
- Keyboard event detection demo section

This project demonstrates:
- DOM manipulation
- Event handling
- Form validation
- Dynamic content rendering
- Clean UI design with modern CSS styling
